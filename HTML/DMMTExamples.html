DMMT Examples

Rule: Clarity trumps Consistency

Good Example: This example page of TutorialRepublic.
It maintains a clean and minimalist design, prioritizing clarity and simplicity over excessive visual consistency.

Open tutorialrepublic.com

Rule: Keep the Noise Down to a Dull Roar

Good Example: Leaders for Leaders's homepage.
It focuses on a single clear message and uses minimal visuals and concise text to avoid overwhelming the user with excessive noise.

Open leadersforleaders.ca

Rule: Trunk Test

Good Example: Mathguide's Elements of Effective Design page.
It displays essential information such as the title, breadcrumbs, and index, allowing users to quickly evaluate options without the need for additional clicks.

Open mathguide.com

Rule: Break pages up into clearly defined areas

Violation Example: Wayfair.com.
Lack of Visual Hierarchy. Visual Hierarchy arranges and organizes website elements so that visitors naturally gravitate toward the most critical elements. The problem with Wayfair.com is that almost every element is the same size and color and has similar copy and icons. So users are being told to shop for beds and mattresses, vanities, area rugs, outdoor seating, and bedding sets at the same time. This can be overwhelming for the user and rather than deciding where to click first, some users might leave the site.

Open wayfair.com

Rule: Format Text to support Scanning

Violation Example: 007 Museum Sweden.
This website is all over the place with images and text thrown about with no thought at all for the user. The text is hard to read and hard to find, and is a fantastic example of how NOT to format a website

Open 007museum.com

Rule: Omit Needless Words

Violation Example: gatesnfences.com.
The problem with Gates N Fences's website is that too much information is given to the user at once. This can lead to user paralysis and makes visitors instead choose websites with simpler, less confusing designs.

Open gatesnfences.com
